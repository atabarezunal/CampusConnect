📌 Nombre del proyecto

CampusConnect

📖 Descripción

Sistema distribuido basado en microservicios que permite a estudiantes universitarios:

crear grupos de estudio
compartir recursos
colaborar en proyectos
recibir notificaciones
🏗️ Arquitectura
API Gateway: Laravel
Microservicios:
Laravel (Auth)
Django (Users)
Express (Study Groups)
Flask (Projects)
Express (Notifications)
🗄️ Bases de datos
MySQL (Monolito/Auth)
PostgreSQL (Users/Projects)
Firebase (Study Groups)
Firebase (Notifications)
